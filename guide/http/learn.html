# 神奇的URI

在很多时候，互联网上的URL和URI几乎都是等同的，其实二者会有一些很细微的差别。

URI包括URL和URN

他们的区别如表

名称 英文简称 备注
URI Uniform resource identifier 包括了URL和URN
URL Uniform resourse location 统一资源定位符，与位置有关
URN Uniform resourse name 统一资源名，与资源所在地无关，如urn:ietf:rfc:2141

按照正常的逻辑来看，URI更有利于表示资源所属位置的关系，而按照《HTTP权威指南》来看，可能在不久的将来，URN的使用将会越来越广。

基础的URL由几个部分组成：

分别为：

protocol😕/host:port/file?query#flag

这样的形式遍布。

  1. 根据过去的经验总结，往往port（端口）是省略的

按照默认来说

http协议的默认端口为80

https协议的默认端口为443

  1. 而file部分有时候也可以被忽略，由于web服务器自带的默认文件名。
